## Project Name: BudgetBazaar

### Description:
BudgetBazaar is an online e-commerce platform that specializes in offering a wide range of affordable products and goods to budget-conscious shoppers.

### Technologies Used:
- Java
- Spring Boot
- Spring Data JPA
- MySQL Workbench
- Maven
- Spring Web

### Features:
* Show a list of products
* Add products
* Update products
* Delete products
